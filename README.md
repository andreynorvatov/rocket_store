## Цели и задачи проекта
* Реализовать рабочую модель онлайн-магазина на базе фреймворка Django, на практике освоить работу с запросами и ответами, настройку базы данных, создание и работу с моделями, формами, представлениями, шаблонами, статическими файлами.

* Выполнить фронтенд сайта на HTML и CSS, без использования фронтэнд-фреймворков.

* Развернуть работоспособный учебный проект на [Heroku](https://www.heroku.com).

## Описание
В интернет-магазине пользователь может просматривать каталог товаров, добавлять товары в корзину, изменять количество и удалять товары из корзины. Также доступна форма оформления заказа. Реализация осуществляется через создание моделей каталога товаров, добавление на сайт администрирования, создание представлений. Корзина для покупок реализована через сессии Django для хранения данных для каждого посетителя.

Фронтенд часть сайта выполнена адаптивной (медиа запросы на 950px, 710px, 499px) с использованием Flex-box. Основной шаблон **base.html** расширяют шаблоны остальных приложений (список товаров, корзина).
